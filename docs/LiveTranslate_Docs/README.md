# MLingo Platform Notes

Real-time speech translation and AI language companion for macOS.

These files are the original concept notes. The canonical implementation design and milestone status live under `docs/superpowers/`.

## Vision
Build a native Swift application that captures system audio, performs local speech recognition, translates speech with low latency, displays subtitles, optionally performs AI voice dubbing, and evolves into an extensible AI language platform.

## Product Roadmap
1. Live Subtitle
2. Live Translation
3. Live Voice Translation
4. Knowledge Engine
5. AI Companion

See the docs/ folder for detailed design.
